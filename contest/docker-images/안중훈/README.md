# 같이 써보아요
### 1. [도커 허브](https://hub.docker.com/repository/docker/deadricecake/xams-tree/general)에서 이미지 땡겨오기
```
docker pull deadricecake/xmas-tree/latest
```
### 2. 땡겨온 이미지를 통해 컨테이너 실행
```
docker run -it deadricecake/xmas-tree/latest
```
